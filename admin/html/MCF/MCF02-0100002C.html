[SMS등록방법]

  • 직접 작성방법
    1. 변수설정
    2. 메시지 내용 입력
    3. 입력 중 설정한 변수 개수와 동일하게 맞춤
    • 예시) 포인트 지급직접 입력
      1. {name}: 회원명 변수
      2. {point}: 포인트 변수
      3. {url}: 경로 변수
  • 엑셀 업로드 작성방법
    1. 템플릿양식 다운로드
    2. 다운로드 양식에 포함될 내용(변수포함) 입력
    3. 저장 후 Excel 셀과 동일하게 변수설정
    4. 메시지 입력 중 설정한 변수 개수와 동일하게 맞춤

[영리 목적의 광고성 정보 발송 안내]

  • 필수정보 안내 목적의 "주문접수 안내"와 같은 SMS는 수신동의 없이 보낼 수 있지만 영리 목적의 광고성 정보 발송 시에는 반드시 수신동의한 회원에게만 발송해야 합니다.
  • 반드시 수신동의를 받은 회원에게만 발송 해야 합니다.
  • 오후 9시 부터 그 다음날 오전 8시에는 별도의 사전동의를 받은 경우가 아니면 발송해서는 안됩니다.
  • SMS 내용에 전송자의 명칭/연락처/수신거부기능(수신거부방법과 수신자가 비용부담하지 않는다는 내용(예시) 무료수신거부 080-000-0000))이 포함되어야 합니다.
  • SMS내용 시작 부분에 (광고)를 명시해야 합니다.
  • 위 내용을 위반하는 경우 3천만원 이하의 과태료가 부과 됩니다.
