[SMS등록방법]

  • 직접 작성방법
    1. 변수설정
    2. 메시지 내용 입력
    3. 입력 중 설정한 변수 개수와 동일하게 맞춤
    • 예시) 포인트 지급직접 입력
      1. {name}: 회원명 변수
      2. {point}: 포인트 변수
      3. {url}: 경로 변수
  • 엑셀 업로드 작성방법
    1. 템플릿양식 다운로드
    2. 다운로드 양식에 포함될 내용(변수포함) 입력
    3. 저장 후 Excel 셀과 동일하게 변수설정
    4. 메시지 입력 중 설정한 변수 개수와 동일하게 맞춤
